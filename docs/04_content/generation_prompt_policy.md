# 생성 프롬프트 정책 템플릿

## 목적
- 콘텐츠 생성 품질과 안전성을 일관되게 유지한다.

## 입력 규칙
- 입력 필수값: 레슨 목표, 난이도, 언어, 길이 제한
- 금지 입력: 개인정보, 민감정보, 저작권 침해 요청

## 출력 규칙
- 문장 길이: (TODO)

- 금지: 차별/혐오/의학·법률 오진 유도 표현
