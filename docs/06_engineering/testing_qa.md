# 테스트/QA 템플릿

## 테스트 전략
- 단위 테스트: 핵심 로직 검증
- 통합 테스트: API/데이터 흐름 검증
- E2E 테스트: 핵심 학습 루프 검증
- 수동 QA: UX/카피/접근성 최종 확인

## MVP 필수 테스트 시나리오
- [ ] 홈에서 오늘의 레슨 시작
- [ ] 레슨 카드 2장 스와이프 완료
- [ ] 연습/퀴즈 완료 후 리포트 진입
- [ ] 레슨 저장/해제
- [ ] TTS 재생 성공/실패 처리
